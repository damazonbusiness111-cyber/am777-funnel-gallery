FAQ

Common questions.

Do I need an existing website?
No. Each template is a standalone page published on GitHub Pages. No existing site required.
Is pricing shown on this page?
Pricing is shared after inquiry. Message on WhatsApp or use the form to get a recommendation first.
Is the chatbot a real AI agent?
Templates include chatbot-style inquiry flows. Advanced AI receptionist is scoped separately per project.
Is a CRM dashboard included?
Google Sheets record sync is available as an add-on. Full CRM dashboard is a separate scoped setup.
Is custom domain included?
A free GitHub Pages link is included. Custom domain configuration is quoted separately.
What if I need something different?
Use the Custom Template option or send an inquiry. AM777 scopes custom logic and workflows on request.
